¿Qué es?

Es el proceso de cuidar, revisar y proteger los equipos y programas informáticos para que funcionen correctamente, de manera segura y sin interrupciones. Incluye actividades técnicas y preventivas que ayudan a evitar fallas, mejorar el rendimiento y proteger la información almacenada en el sistema.

Este mantenimiento se realiza tanto en el hardware (parte física de la computadora) como en el software (programas y sistemas). También abarca las redes, servidores y bases de datos que permiten la comunicación y el funcionamiento de los sistemas informáticos.

El aseguramiento de la operación consiste en garantizar que todos los componentes del sistema trabajen de forma continua, eficiente y segura. Para ello se realizan actualizaciones, respaldos de información, monitoreo del sistema y medidas de seguridad informática.

Tipos de Mantenimiento

Existen diferentes tipos de mantenimiento de sistemas informáticos que permiten garantizar su correcto funcionamiento. Cada uno cumple una función específica, ya sea prevenir fallas, corregir errores, anticipar problemas o mejorar el sistema. La aplicación conjunta de estos mantenimientos asegura un mejor rendimiento, mayor seguridad y una vida útil más prolongada de los equipos.

    Preventivo: Es el más importante, porque evita problemas antes de que ocurran.

    Caracteristicas:
  • Se realiza de forma planificada y periódica
  • Reduce costos de reparación
  • Disminuye el riesgo de fallas graves

    • Actividades comunes
  • Limpieza interna del CPU (para evitar sobrecalentamiento)
  • Actualización del sistema operativo
  • Revisión de discos duros
  • Escaneo antivirus
  • Optimización del sistema

    • Correctivo: Se aplica cuando el sistema ya presenta fallas.

    Caracteristicas:
  • No es planificado (ocurre cuando algo falla)
  • Puede ser urgente
  • Puede implicar costos altos

    • Tipos dentro del correctivo:
  • Correctivo inmediato: se arregla en el momento
  • Correctivo diferido: se programa para después

    • Actividades:
  • Reparación de componentes dañados
  • Eliminación de virus o malware
  • Recuperación de datos
  • Reinstalación del sistema operativo

    • Predictivo: Analiza el sistema para prevenir errores futuros.

    Caracteristicas:
  • Se basa en datos y monitoreo
  • Usa software especializado
  • Ayuda a evitar fallas inesperadas

    • Ejemplos:
  • Programas que revisan la temperatura del equipo
  • Análisis del estado del disco duro (SMART)
  • Monitoreo del rendimiento del sistema

    • Mantenimiento Evolutivo (más a fondo)Se enfoca en la mejora continua del sistema.

    Características:
  • No corrige fallas, mejora el sistema
  • Responde a nuevas necesidades
  • Mantiene el sistema actualizado

    • Actividades:
  • Instalación de nuevas aplicaciones
  • Actualización de hardware (RAM, disco SSD)
  • Mejora de seguridad
  • Adaptación a nuevas tecnologías

Seguridad del Sistema

La seguridad del sistema informático es el conjunto de medidas, técnicas y herramientas que se utilizan para proteger los equipos, programas, redes y datos frente a accesos no autorizados, ataques, daños o pérdidas de información.

  • Confidencial (solo acceden personas autorizadas)
  • Integra (no sea modificada sin permiso)
  • Disponible (pueda usarse cuando se necesite)

    • Esto se conoce como el modelo CIA (Confidentiality, Integrity, Availability).

    ¿Qué protege la seguridad informática? La seguridad no solo protege una computadora, sino:

  • Datos personales y archivos importantes
  • Sistemas operativos
  • Programas y aplicaciones
  • Redes (WiFi, internet)
  • Servidores y bases de datos

    Importancia del mantenimiento y la seguridad de sistemas informáticos

    El mantenimiento y la seguridad de los sistemas informáticos son fundamentales porque permiten que los equipos, programas y datos funcionen de manera eficiente, segura y continua. Sin estas prácticas, los sistemas pueden fallar, volverse vulnerables o dejar de ser útiles.

    🔧 Importancia del mantenimiento

    El mantenimiento es importante porque:

    ⚙️1. Evita fallas inesperadas

  • Permite detectar problemas antes de que ocurran, evitando interrupciones en el trabajo.

    • 🚀 2. Mejora el rendimiento

  • Un sistema bien mantenido funciona más rápido y de forma más eficiente.

    • 💻 3. Prolonga la vida útil

  • Los equipos duran más tiempo si reciben mantenimiento constante.

    • 💰 4. Reduce costos

  • Prevenir es más barato que reparar o reemplazar equipos dañados.

    • 🔄 5. Garantiza continuidad

  • Evita que empresas o usuarios pierdan tiempo por fallos técnicos.
